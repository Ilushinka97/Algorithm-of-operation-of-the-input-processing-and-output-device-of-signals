# Algorithm-of-operation-of-the-input-processing-and-output-device-of-signals
Алгоритм функционирования устройства ввода, обработки и вывода сигналов на основе микроконтроллера ADuC812.

Разработанная программа реализует следующие задачи:
- Измерение частоты входного меандра и генерацию выходного ШИМ-сигнала, пропорционального частоте входного меандра. 
- Прием значений в диапазоне от 0 до 180, передаваемых по интерфейсу UART, и изменение угла открытия тиристоров путём изменения напряжения, до которого заряжается конденсатор в генераторе пилообразного напряжения. Как следствие, производится изменение момента подачи управляющих импульсов пропорционально тому значению, которое приходит по UART.
- Передача характеристик входных сигналов по интерфейсу RS-232 в соответствии с протоколом Modbus RTU. 
- Измерение частоты меандра. Для этого используется внешнее прерывания INT0, которое настроено на срабатывание по заднему фронту сигнала. В обработчике данного прерывания инкрементируется счетчик counter_m. Таким образом подсчитывается количество импульсов. Далее происходит вывод ШИМ-сигнала, пропорционального полученной частоте меандра, обеспечивающегося с помощью встроенного в ADuC842 ШИМ-генератора.  

Реализация передачи характеристик входных сигналов по протоколу Modbus RTU организована с помощью имеющегося в ADuC842 встроенного UART. В соответствии с заданием, UART настроен на обмен данными на скорости 9600 бит в секунду. Реализованы варианты передачи как обеих характеристик входных сигналов, так и каждой в отдельности, а также вариант отправки сообщения об ошибке при некорректном запросе.
